# Projet d'intégration de maquette Figma

Ce dépôt contient une application [Next.js](https://nextjs.org) pour l'intégration d'une maquette UI/UX fournie depuis Figma.

## Démarrage rapide

1. Installer les dépendances :

```bash
pnpm install
```

2. Lancer le serveur de dev :

```bash
pnpm dev
```

3. Ouvrir [http://localhost:3000](http://localhost:3000).
